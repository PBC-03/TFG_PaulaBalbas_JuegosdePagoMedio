Algoritmo basado en Juegos de Energía
=====================================

Algoritmo que resuelve juegos de pago medio utilizando el enfoque de **juegos de energía**. Donde se reescala el digrafo original de forma que, en cada iteración,
el problema pase a ser uno de síntesis de estrategias. En cada reescalado, se computa un juego de energía para determinar si el valor del juego
se manetiene no negativo para cada nodo.
