Documentación del TFG de Paula Balbás Corbacho

Esta documentación recoge y explica el funcionamiento del código desarrollado para el Trabajo de Fin de Grado. El proyecto se centra en la resolución de juegos de pago medio mediante distintos algoritmos. En particular, se describen:

  • Juegos de energía: Enfoque basado en técnicas de reducción y estrategia iterativa.

  • Algoritmo de Zwick y Paterson: Algoritmo clásico utilizando de computar valores y estrategias óptimas.

Por otro lado, dentro del código juegos de energía, se proporciona un algoritmo que resuelve juegos de pago medio utilizando el enfoque de juegos de energía.

Cada módulo contiene tanto las funciones, clases y estructuras utilizadas, como sus respectivas explicaciones, extraídas automáticamente desde los comentarios del código.
